Today, while walking from the the entrance to Ramage Peak trail near the public golf course near Redwood Road in what seems to be the northernmost point of Castro Valley, California, I felt more motivated than usual to stare at some arbitrary point in front of me instead of looking all around me because I wanted to be as focused as possible on whatever my train of thought was. For the first time in what seems like years, I feel the need to take a break from using marijuana (because I think that chronic heavy use of marijuana interfered with my short term memory and made me excessively anxious). I was in a rush to get out of Castro Valley and into a less population dense and more convenient place to use my laptop: Dublin/Pleasanton California. After taking the BART train (Bay Area Rapid Transit) from Castro Valley station to Dublin/Pleasanton (east) station, I got some healthy plant-based munchies from the nearby Whole Foods Market and then set up my tent next to my favorite bush in that wide open field south of the BART station and east of the 580 freeway. For several hours, I got super cozy and concentrated on my website update process and ended up trimming down my primary website (Karlina Object dot WordPress dot Com) down to a very clean new beginning, backed up each of its pages and the source code and media files on GitHub to the WayBack Machine at Archive dot Org, and got my secondary website (Karbytes For Life dot WordPress dot Com) up to date and backed up to the WayBack Machine also. (I also removed some social media posts on my Twitter, Minds, and Instagram accounts which seemed a bit negativistic to me so that only positive or neutral content remains).

For each web page besides the START_PAGE of my primary website, I added the following highlighted text to the web page body in order to signify the timeframe in which I expect to have that web page fully up to date:

This web page is currently undergoing construction and will tentatively be up to date no later than 01_AUGUST_2022.

(Of course, I do not intend to take that long to update those pages (and I intend to include more than the eight pages which are listed on the START_PAGE after the first eight pages are fully up to date). I just said that I would take approximately two weeks to do what I know I can realistically get done this weekend if I just stay in Dublin and treat this weekend like a website reboot boot camp opportunity).

I am sitting in that little jungle area where people feed stray cats and provide them little cat houses to live in on the north side of the Whole Foods Market to use its Wi-Fi (because it is more reliable than any Wi-Fi network I have tried in Castro Valley). What is fascinating to me is that downloading the Kubuntu 22.04 LTS iso file only took about ten minutes while that same file would have taken approximately three hours using the Wi-Fi at my legal residence in Castro Valley. (That Wi-Fi is known to cause me upload errors and connection interuptions to the extent that I have decided that I cannot rely on that network for anything but “emergency” Internet usage).

Anyway, I am about to use the Rufus tool to create a bootable USB with the Kubuntu Linux operating system on it so that I can either use that USB stick as an alternative operating system without having to install it on my HP laptop which has Windows 10 as its native operating system and/or I can partition the laptop’s hard drive to make way for that Kubuntu operating system (which may corrupt my laptop and render it useless). Before I risk losing this laptop, I would like to back up this note to my secondary website so that I do not forget the events which transpired this afternoon and tonight.

Why would I go to the trouble to install Kubuntu on my Windows laptop? I want to use the Unix command line terminal to compile and run C++ programs and I want to promote using open source software as much as possible. Open source software is free for anyone to use, make copies of, sell, and to incorporate into other pieces of intellectual property. My websites are open source (which I consider to be almost synonymous with public domain). I prefer the term public domain because public domain insinuates that the intellectual property it describes is in its final state while also being open source (and public domain is what a lot of pieces of intellectual property are labeled after their original authors die. I like to live as though I am already dead and my job is to curate the intellectual property left behind by that dead person as that dead person’s appointed heir).
