The above tutorial suggested that I use balenaEtcher (which is compatible with Linux, Windows and Mac OS) instead of Rufus (which is only compatible with Windows).

https://www.balena.io/etcher/

After downloading a Linux distibution named Kubuntu (as a file whose extension is .iso), I was able to easily use balenaEtcher to burn the iso file onto a USB flash drive such that the flash drive is solely used to either install Kubuntu onto a machine or else allow the machine to run Kubuntu from the flash drive without installing Kubuntu onto the machine.

I went ahead and installed Kubuntu onto my laptop (which apparently deleted the native Windows 10 operating system).

Once I had Kubuntu running and an Internet connection established using my Android phone to establish a USB wired connection, I tried to use the command line (i.e. Konsole) to install the Sublime Text editor application (but doing so caused errors which seemed to prevent me from being able to make any further updates via the command line. To mitigate the problem, I re-installed the operating system using my handy dandy purple USB flash drive. Then I used the Discover application store to download Sublime Text via a graphical user interface instead of the command line with no issues).

Then I used Sublime Text to create a simple “Hello World” C++ application. I was able to install and run the C++ compiler g++ to compile my test.cpp source code into an executable application named app using the following Unix console command:

g++ test.cpp -o app

Then I entered the next command to see the message “Hello World!” appear on the command line console as program output:

./app

So far, I appear to have everything I need to get Karlina Object dot WordPress dot Com up to date!
